# Proyecto 7- Best vehicles features
Proyecto 7 
La visualicación y la interacción activa con información es una herramienta útil para dar a conocer datos de interés a usuarios o clientes. La creación de una aplicación, Best vehicles features, realizada en VisualCode con Pandas, Streamlit, Plotly; que permite al usuario indagar sobre las opciones que existen para elegir el vehiculo de su referencia.
El boton de histrograma muestra la distribución por modelo y condicion y el boton de dispersión le dara un panorama de acuerdo a la transmision y el recorrido de diferentes vehiculos.
